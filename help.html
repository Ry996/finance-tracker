How to Use

  1. Go to Add to record an income or expense.
  2. Return to Home to see overview and recent records.
  3. Open Stats to view monthly charts (Canvas).
  4. Use Records to filter and manage all entries.
  5. Use Categories to add custom categories.

Accessibility Notes

  • Form inputs have labels and grouped controls (fieldset/legend).
  • Status messages use aria-live for screen readers.
  • Keyboard navigation supported via focus styles.

FAQ

Where is my data stored? In your browser’s LocalStorage.

Will data sync across devices? No. It stays on the current browser/device.
